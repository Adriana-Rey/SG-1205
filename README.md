# CQ Control - SG-1205

Aplicativo local para consulta e acompanhamento das tarefas do arquivo `SG-1205.xlsx`.

## Como abrir

Abra o arquivo `index.html` no navegador.

## Recursos

- Painel com indicadores de tarefas e pendências.
- Filtros por TAG, equipamento, responsável e situação.
- Busca por TAG, item ou descrição.
- Edição dos controles de qualidade e do responsável.
- Inclusão de até quatro fotos por tarefa, com pré-visualização e remoção.
- Campo de observação individual por atividade.
- Persistência das alterações no navegador (`localStorage`).
- Exportação dos registros filtrados em CSV.

O aplicativo não altera a planilha original. O comando **Restaurar original** remove as alterações locais de uma tarefa.
